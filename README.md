# HtmlPieces
Pieces of browser pages (css + js + demohtml + generator functions)

I try to create some substructures for use in browser pages.
These should be embeddable in any html pages, without 
to disturb or to be disturbed by surrounding or contained parts

The complete isolation - because of the inheritance strategies of css -
has not yet succeeded. It may be that this will not succeed with reasonable
effort.

Current attempts:

	The notebook in different implementation variants.
  
		One of the variants (snb) is realized via styles and corresponding Html only.
    The generator function is of course Javascript. 

    The other variants are based on the new styles grid and flex.
    Here again, the js content is not particularly high.

    The demopage containes functions for moving the notebook-tabs to other
    sides (of the notebook).
